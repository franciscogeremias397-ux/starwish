# Requirements

## Goal

根据《星空银河需求文档.md》开发《银河回信》微信小程序，优先保证沉浸式星空视觉、飞升仪式感和心愿星交互体验。

## Functional Requirements

- 首页呈现深蓝黑星空、银河、背景星和可点击的用户心愿星。
- 用户可输入心愿、烦恼或心事，当前输入限制为 200 字。
- 点击“飞升”后展示斜向上的流星飞升动效，并生成新的心愿星。
- 点击心愿星后查看对应星语详情。
- 支持“我的星星”、设置和基础本地演示数据。

## Non-Functional Requirements

- 保持高级、安静、神秘、治愈的视觉气质。
- 用户心愿星必须和普通背景星明显区分。
- 不提交密钥、环境变量、私钥或本地预览产物。
- 不在未确认需求时新增后端、云函数、举报、审核或额外弹窗能力。
